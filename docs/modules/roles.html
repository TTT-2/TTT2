Functions

GenerateNewRoleID ()
Generates a new subrole id. starts with 1 to prevent incompatibilities with ROLE_ANY => new roles will start at the id: 7
  • 0 = ROLE_INNOCENT
  • 1 = ROLE_TRAITOR
  • 2 = ROLE_DETECTIVE
  • 3 = ROLE_ANY
  • 4, 5, 6 = nop

Returns:

    number new generated subrole id
